# call-java

## 快速开始

### 1. 安装 kona JDK

```bash
bash install-kona-jdk.sh
source ~/.bashrc
```

### 2. 编译生成字节码

```bash
cd java
javac HelloWorld.java
cd -
```

### 3. 运行程序
```bash
cargo run
```
